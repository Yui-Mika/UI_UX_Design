## 🧩 Required Bootstrap Components Used

To meet and exceed the assignment requirements, I utilized the **Bootstrap Grid System** along with the following **8+ Bootstrap Components**:

1.  **Navbar:** Fully responsive navigation bar that collapses into a hamburger menu on smaller screens.
2.  **Carousel:** A dynamic image slider used in the Hero section to showcase bakery highlights.
3.  **Cards:** Used to elegantly display individual cake products with images, titles, and descriptions.
4.  **Badges:** Added "New" and "Hot" tags on product cards to highlight specific items.
5.  **Buttons:** Styled call-to-action buttons (e.g., "Order Now", "See Menu").
6.  **Accordion:** Implemented an interactive FAQ section that expands and collapses.
7.  **Form:** A complete and styled "Contact Us" form with input fields and a submit button.
8.  **Footer:** A clean, informative footer area at the bottom of the page.
9.  **Grid System (`container`, `row`, `col`):** The core structure holding all sections together responsively.
